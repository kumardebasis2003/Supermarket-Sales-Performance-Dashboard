# 📊 Supermarket Sales Performance Dashboard

## 🚀 Overview
This project is a Power BI dashboard that provides insights into supermarket sales performance.

## 📌 Key Features
- Total Sales, Profit, and Profit Percentage
- Top Selling Product Analysis
- Sales Trend by Month
- Customer-wise Sales Analysis
- Segment-wise Sales Distribution
- Country-wise Sales Performance

## 🛠 Tools Used
- Power BI
- Excel (Dataset)
